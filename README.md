# Odin Flexbox Project

## Overview
This project is a landing page built as part of [The Odin Project](https://www.theodinproject.com/).  
It demonstrates layout and styling using **Flexbox**, responsive design, and basic web development principles.

## Features
- Fully responsive layout using **Flexbox**
- Header with logo and navigation menu
- Hero section with content and image
- Features section showcasing content boxes
- Testimonial section
- Call-to-action (CTA) section
- Footer with copyright information

## Technologies Used
- HTML5
- CSS3 (Flexbox)
- Google Fonts (Inter)
- Git & GitHub for version control
